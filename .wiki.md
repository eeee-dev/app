# Project Summary
The Financial Management Dashboard for the ë Ecosystem is a comprehensive platform designed to optimize financial operations for businesses. It offers functionalities such as budget management, expense tracking, income reporting, and invoice processing, ensuring compliance with regulations like Mauritius VAT. By utilizing automation and analytics, the application improves operational efficiency and facilitates informed decision-making. Built on Supabase, it provides a reliable and user-friendly environment for managing financial activities.

# Project Module Description
- **Dashboard**: Displays key financial metrics and user activities.
- **Expenses**: Manages expenses with CRUD operations and VAT calculations.
- **Income**: Tracks income through manual entries and invoice management.
- **Invoices**: Facilitates the creation and tracking of invoices.
- **Departments**: Manages department details and budget allocations.
- **Settings**: Configuration options for user preferences and notifications.
- **Reports**: Generates and analyzes financial reports.
- **Bank Statements**: Uploads and reconciles bank transactions.
- **OCR Integration**: Real-time invoice scanning and data extraction.
- **Bulk Import**: Supports bulk importing of financial data.
- **Notifications**: Alerts users about important updates.
- **Statistics**: Displays uploaded screenshots with filtering and search functionalities.
- **Projects**: Manages project details, budgets, and statuses.
- **Tax Return**: Ensures compliance with Mauritius VAT regulations.
- **User Access**: Authentication for user access to the application.
- **Audit Trail**: Monitors system activities and user actions.
- **Password Reset**: Secure password resets for users.
- **Login**: Enhanced branding with the ë logo on the login page.
- **Setup Script**: Automates setup, including database tables and category creation.
- **Database Setup**: SQL scripts for creating necessary tables and security policies.

# Technology Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL)
- **Build Tool**: Vite 6
- **State Management**: React Query (TanStack Query)
- **Routing**: React Router v7
- **Form Handling**: React Hook Form
- **Charts**: Recharts
- **OCR**: Tesseract.js
- **Data Import**: Python (for parsing Excel files)

# Usage
1. **Install dependencies**: 
   ```bash
   pnpm install
   ```
2. **Configure Environment Variables**: Create a `.env` file in the project root with necessary Supabase credentials.
3. **Run SQL Migration**: 
   - Open the Supabase SQL Editor and run the migration script located at `/workspace/fresh-dashboard/final-setup.sql`.
4. **Run Setup Script**: 
   ```bash
   node setup-complete.mjs
   ```
5. **Run Database Installation**: 
   ```bash
   python3 install_database.py
   ```
6. **Build for Production**: 
   ```bash
   pnpm run build
   ```
7. **Deploy the Application**: 
   - Follow the instructions in **DEPLOYMENT.md** for deployment steps.
   - **Publish the Application**: Click the “Publish” button in the MGX platform to make the application live.
8. **Run Development Server**: 
   ```bash
   pnpm run dev
   ```
